About the app:

The purpose of this application is to allow operators to create the route that a robot needs to follow for conducting inspections on an industrial site, while also enabling them to visually track its movements through a simulator.

How to use the app:

  • Click on the "hamburger" button in the top-left corner of the screen and select "Planner" in order to create a new route.
  • Click on the "hamburger" button in the top-left corner of the screen and select "Player" in order to visualise the robot's movements.

Application structure:

The application is structured into modules that represent different features. A module can in turn contain several modules of various types, each with well-defined purposes:
  1. data-access - contains code for interacting with a back-end system. It also includes all the code related to state management.
  2. feature - contains a set of files that configure a business use case or a page in an application
  3. ui - contains only presentational components (also called "dumb" components).
  4. util - contains low-level utilities used by many modules.
  • The app folder contains the root component and the application's routing configuration.
  • The shell folder contains wrappers for different feature modules.
  • The home folder exclusively contains components, services, models, etc., dedicated to the functionality of the Home feature.
  • The planner folder exclusively contains components, services, models, etc., dedicated to the functionality of the Planner feature.
  • The player folder exclusively contains components, services, models, etc., dedicated to the functionality of the Player feature.
  • The assets folder holds static files like images and JSON data for direct use in the application.

Application hierarchy:

Application setup:

To serve, build, and run the Angular app successfully, please follow these steps:
  1. Make sure you have Node.js and npm installed on your system. You can download and install them from Node.js official website.
  2. Install Angular CLI globally by running the following command in your terminal: npm install -g @angular/cli
  3. Install Dependencies by running the following command in your terminal: npm install
  4. Serve the App locally by running the following command in your terminal: ng serve
  5. Build the App by running the following command in your terminal: ng build
For more information, please refer to the README file located in the root of the repository.
